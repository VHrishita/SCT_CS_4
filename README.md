# Keylogger Simulator

A fully client-side Keylogger Simulator built using HTML, CSS, and JavaScript.
This project is intended to help developers understand keyboard event handling, input monitoring, and secure data storage in a safe, controlled environment.

---

## Features

- Capture keyboard input inside the browser (local testing only)
- Display captured keystrokes in real-time
- Store keystrokes temporarily in memory for demonstration
- Option to clear captured data
- Fully client-side: no data leaves your browser

---

## How It Works

- Listens for keyboard events (keydown / keyup)
- Records characters pressed in a local array
- Updates the live display dynamically
